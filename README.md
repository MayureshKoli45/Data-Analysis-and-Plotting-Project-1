# Data-Analysis-and-Plotting-Project-1

Analysing How the temperature changes on each day of the year from 2005-2014 VS 2015 and scattering the record of maximum and minimum temperature broken by 2015 on each day of a year of the region Michigan,USA.

Also cleaning the data by using numpy and pandas and also plotting the graph by using matplotlib.

The data csv file, jupyter notebook source code and plot png file are provided.
